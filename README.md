Create a virtual environment 
```python3 -m venv venv```

Install spacy and pandas if not yet
```pip install spacy pandas google-cloud-translate dotenv```


Install spaCy's italian model:
```python -m spacy download it_core_news_sm```

Uses this list, created by Matthias Buchmeier:
https://en.m.wiktionary.org/wiki/User:Matthias_Buchmeier/Italian_frequency_list-1-5000
